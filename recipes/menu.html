# 菜单食谱

# 爆炒郡肝（猪猪最爱🚩）

主要食材

  • 郡肝✔️
  • 泡椒✔️
  • 二荆条（辣椒）✔️

配料

大葱、姜、蒜、料酒、豆瓣酱、淀粉、耗油

步骤

  1. 郡肝清洗干净切片或改花刀，加盐➕料酒➕清油➕豆瓣酱➕花椒➕蒜片腌制10分钟。
  2. 准备斜刀葱段➕二荆条切片➕姜蒜切片备用
  3. 锅中热油（稍微多点儿），先放入郡肝翻炒至变色，再加入葱段，二荆条继续翻炒
  4. 加入些许盐➕鸡精➕耗油
  5. 翻炒五六七八下出锅🎉

# 辣椒炒肉

主要食材

  • 猪肉✔️
  • 青红椒（宜选彩椒，猪猪不太能吃辣）✔️

配料

  • 姜（猪猪不吃可不放）、蒜、生抽、耗油、老抽🎉

步骤

  1. 猪肉切片，辣椒切块备用，准备蒜末
  2. 热锅倒入少许菜油，放入猪肉片煸炒出多余油脂，盛出备用
  3. 锅里留油，放入蒜末爆香，倒入辣椒翻炒均匀，加一勺生抽➕半勺老抽➕少许盐调味，爆炒出锅🎉

# 鱼香肉丝

主要食材

  • 猪肉里脊✔️
  • 青椒✔️
  • 木耳✔️
  • 胡萝✔️

配料

生抽、老抽、豆瓣酱、耗油、白糖、淀粉、料酒、水、葱、蒜、醋

步骤

  1. 木耳、胡萝卜、青椒、猪肉切丝备用🎉
  2. 猪肉丝腌制：盐➕一勺生抽➕一勺料酒➕一勺淀粉抓匀
  3. 调料汁：两勺生抽➕两勺醋➕一勺耗油➕一勺白糖➕一勺淀粉➕半碗水，搅拌均匀
  4. 起锅烧油，放入肉丝炒变色盛出备用，锅底留油葱蒜炒香，加入些许豆瓣酱(猪猪不吃就不加)
  5. 倒入尖椒、胡萝卜、木耳炒至断生，放入肉丝，淋上料汁，翻炒均匀出锅🎉

# 肝腰合炒

主要食材

  • 猪肝✔️
  • 猪腰子✔️
  • 泡椒✔️
  • 青红椒✔️

配料

大葱、蒜、泡姜、料酒、花椒、淀粉、生抽、豆瓣酱🎉

步骤

  1. 猪腰从中间破开，出去腰骚，切花刀（先斜切45°，再三刀一垂直）
  2. 猪肝切薄与猪腰放一起，加葱姜料酒➕生抽➕淀粉腌制十分钟
  3. 准备泡椒切段、泡姜切片、葱切断
  4. 起锅烧油放入泡椒、泡姜、葱段、蒜片、豆瓣酱、花椒爆香
  5. 放入猪肝和猪腰快速爆炒，变色后可放入自由搭配的配菜，翻炒均匀出锅🎉
